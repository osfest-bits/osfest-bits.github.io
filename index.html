What is OS Fest?

Os fest means Open Source Fest and open source means every thing for a newly started programmer who is setting his path in this competitive world of coding. The open source movement is the reason that technology has developed at such a breakneck pace for the past few decades.So don't you think open source deserves some celebration of its own for this great contribution in our day to day life?


We are celebrating a festival dedicated specially for open source to make understand the concept to the newbies who are setting their way in this competitive world and contributing in open source community so that this world always remains open for all the generation of people.

To make new friends

With over 100 developers and designers gathering from all over the city in BITS Edu Campus, grab your chance of connecting with people to discuss ideas and opportunities. With Exclusive Networking with the GitHub & Mozilla Communities.

To learn

Mentoring beginners is always helpful for them and rewarding for us. We’re going to have mentors from a number of different organizations present at the venue. You would be take to a ride into opensource world by our awesome mentors.

To build

With this awesome gathering of opensource enthusiast you would learn how to colloborate with other opensource organizations and how to work in team and build real projects and How to Solve Bugs in Real Word problems.

Mastering Git and GitHub

Git is a version control system, but what does that mean? When developers create something (an app, for example), they make constant changes to the code, releasing new versions up to and after the first official (non-beta) release. Version control systems keep these revisions straight, storing the modifications in a central repository. Introduction to GitHub

Browser Extensions

An extension adds features and functions to a browser. It’s created using familiar web-based technologies—HTML, CSS, and JavaScript. It can take advantage of the same web APIs as JavaScript on a web page, but an extension also has access to its own set of JavaScript APIs. This means that you can do a lot more in an extension than you can with code in a web page.
